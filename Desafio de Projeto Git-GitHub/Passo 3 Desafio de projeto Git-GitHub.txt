   *** Criando seu Primeiro Repositório no GitHub Para Compartilhar Seu Progresso**

[Sintaxe Basica Markdow](https://www.markdowguide.org/basic-syntax/)

Primeiros passos 
1 Criar no GitHub online um novo code
2 Clonar na nossa maquina local o código 
>> Comando >>Git clone 
3 ir até a pasta destinada no seu cp e abrir um gitbash 
4 copiar o endereço do cod para o gitbash (terminal estendido para otimizar o uso do Git )
5 para passar um arquivo criado na nasta git dentro do pc
ir em Git status
6 git add. para inserir as novas informações da pasta no git 
depois git commit -m "Inclusão as anotaçõs do curso de Git/GitHub"
8 para commitar o arquivo > git commit -m + " uma descrição em aspas"
9 git status
10 git push origin main
